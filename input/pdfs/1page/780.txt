Résumé. Nous tentons de comparer, dans ce chapitre, les trois approches pour 
étudier des liens vraisemblables entre deux variables binaires, que sont : l’ASI, 
le test de Mac Nemar et le test d’indépendance fondé sur la mesure du χ2. 
Pour ce faire, nous faisons un retour sur des données issues de nos travaux 
passés dans le champ de la didactique des mathématiques. 
1 Introduction 
Comme nous l’avons déjà évoqué succinctement dans la première partie de cet ouvrage, 
la comparaison de deux séries successives de données binaires de type présence-absence ou 
échec-réussite relevées sur le même échantillon d’individus comme cela est le cas en ASI, 
peut aussi être réalisée à l’aide du test du χ² de Mac Nemar.  
2 Le test de Mac Nemar 
2.1 Un exemple introductif issu d’une situation dans le cadre de la

présenté, l’information est résumée dans un tableau 2x2 dont nous donnons un exemple ci-

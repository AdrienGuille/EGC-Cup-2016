Summary
The semantic annotation of resources is at the heart of the semantic Web vision. In fact,
the annotation of resources by meta-data, makes it possible for both human users and software
agents to find the appropriate resources despite being in a large and distributed medium, such as
the Web. However, the annotation process should not be considered as granted, especially with
the growing number of resources when it is almost impossible to assume a manual annotation
process of resources. We propose in this paper, an approach for automatic and semi-automatic
annotation of resources based on the context of citations. The approach presented in this paper
has been experimented and evaluated with the Citeseer database of scientific documents. The
evaluation shows the benefit of using not only the content of resources for the annotation but
also using the citation relationships as context to describe the subjects and themes of resources.
